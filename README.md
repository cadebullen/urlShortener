Spring Boot URL Shortener

A high-performance URL shortener service built with Spring Boot, using Redis for caching and MySQL for persistent storage.

> Features

- Shorten URLs: Convert long URLs into short, base62-encoded strings.
- Fast Redirections: Uses Redis caching to serve popular links instantly.
- Persistent Storage: All mappings are safely stored in MySQL.
- Dockerized: Fully containerized setup for easy deployment.

> Tech Stack

- Java 
- Spring Boot 2.7
- Redis(Cache)
- MySQL(Database)
- Docker & Docker Compose
